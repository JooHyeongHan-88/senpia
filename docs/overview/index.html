Project Overview

웹 기술로 만든 AI Agent 채팅 앱을 단일 .exe 하나로

Svelte(Vite) 정적 자산을 FastAPI 가 서빙하고 PyInstaller 로 단일 EXE 로 패키징하는 로컬 AI Agent 채팅 플랫폼. 다중 대화 세션 · LLM 설정 · 자동 업데이트 · 격리된 확장 시스템 내장. 핵심 컨셉은 "브라우저가 화면이고, EXE 가 서버다" — 사용자는 EXE 를 더블클릭만 하면 로컬에서 백엔드가 켜지고 기본 브라우저에 채팅 화면이 열린다.

3계층 Svelte · FastAPI · PyInstaller 패키지 uv · npm 단일 EXE 배포 onefile 자동 업데이트 sha256 · self-replace 확장 시스템 extensions/
